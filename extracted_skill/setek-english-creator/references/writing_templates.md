# 활동 유형별 서술 템플릿 및 문장 패턴

## 들어가며: 4층 구조의 작동 원리

세특은 500자 내외로 **[진단-전개-해석-전망]** 네 층위를 자연스러운 한 덩어리로 담아야 합니다. 각 층위별 문장 패턴과 활동 유형별 템플릿을 아래에 정리합니다.

---

## 가. 1층 — 진단 (첫 1~2문장)

**목적**: 학생 고유의 영어과 접근 성향·강점을 드러내며 세특 전체의 키를 제시.

### 피해야 할 시작
- ❌ "성실하고 적극적인 학생으로" (범용)
- ❌ "영어 실력이 뛰어나" (상투)
- ❌ "평소 영어에 관심이 많아" (근거 없음)

### 좋은 시작 패턴

**패턴 A — 접근 방식 명시**
> 텍스트를 읽을 때 표면 정보보다 필자의 숨은 가정을 먼저 찾는 독해 성향을 보임.

**패턴 B — 강점 영역 특정**
> 단문의 정확성보다 장문 맥락 안에서 의미를 조율하는 능력이 두드러짐.

**패턴 C — 언어 사용의 태도**
> 영어를 지식이 아닌 생각의 도구로 여기며, 모호한 개념일수록 영어 정의를 우선 찾아 한국어 논의와 대조하는 학습 습관이 있음.

**패턴 D — 학습 맥락 설정**
> 읽기 자료를 자신의 관심 분야(도시 재생)와 연결해 재해석하려는 시도가 2학기 내내 지속됨.

---

## 나. 2층 — 전개 (활동 근거)

**목적**: 2~3가지 구체적 활동을 **무엇을·어떻게·그 결과**까지 포함해 묘사.

### 활동 유형별 템플릿

#### (1) 발표·프레젠테이션

**뼈대**: [소재] + [접근 각도] + [구성 전략] + [관객 반응/산출물]

예시:
> 공정무역을 주제로 한 5분 영어 프레젠테이션에서 소비자 관점과 생산자 관점을 대조하는 투 트랙 구조를 택하고, 통계 그래프의 가시성을 확보하기 위해 슬라이드 레이아웃을 수차례 수정함. 청중 질문 응답에서 준비한 내용 외의 즉흥적 예시를 추가 제시하는 유연성을 보임.

#### (2) 에세이·쓰기

**뼈대**: [주제] + [논증 구조] + [수정 이력] + [언어 사용 특징]

예시:
> AI 생성 콘텐츠의 저작권을 다룬 Opinion Essay(350단어)에서 '창작의 주체성' 개념을 핵심 쟁점으로 설정하고, 초고의 주장-근거 배치를 피드백 이후 반박-재반박 구조로 전면 재조직함. 종속절 활용과 접속 부사의 다양성이 논리 흐름을 명료하게 지지함.

#### (3) 토론·디베이트

**뼈대**: [쟁점] + [입장 구축] + [상호작용 전략] + [사고 확장]

예시:
> Universal Basic Income 찬반 토론에서 찬성 측으로 배정되었음에도 상대 팀 입론의 강점을 인정한 뒤 조건부 찬성으로 입장을 재설정하는 논리적 유연성을 보임. 반박 과정에서 'means-tested welfare'와 'unconditional transfer'의 개념 구분을 명확히 하며 토론의 질을 한 단계 끌어올림.

#### (4) 독서·원서 프로젝트

**뼈대**: [텍스트] + [해석의 각도] + [확장적 연결] + [산출 형태]

예시:
> *The Kite Runner* 수업 프로젝트에서 아프가니스탄의 역사적 맥락을 배경 조사한 뒤 인물의 침묵을 개인 죄책감이 아닌 정치적 침묵의 은유로 읽는 대안적 해석을 에세이로 정리함. 해당 해석은 학급 토론에서 새로운 논점을 촉발함.

#### (5) 듣기·미디어 활용

**뼈대**: [매체·자료] + [수용 전략] + [재구성 방식]

예시:
> 기후 변화 관련 BBC 팟캐스트 3편을 비교 청취하며 화자별 입장을 표로 정리하고, 영국·미국·인도 전문가의 문제 규정 방식이 다른 이유를 지역 경제 구조로 분석하는 자율 학습을 수행함.

#### (6) 문법·언어 구조 학습

**뼈대**: [언어 요소] + [학습 전략] + [적용 사례]

예시:
> 가정법과 화법 표현을 단순 암기가 아닌 텍스트의 어조·화자 태도 분석 도구로 접근함. 뉴스 기사에서 가정법 과거완료의 사용 맥락을 수집한 미니 코퍼스를 구축해 실제 사용에서의 의미 차이를 정리함.

#### (7) 모둠·협력 활동

**뼈대**: [역할] + [조율 방식] + [팀 기여]

예시:
> 지속가능 도시 설계 그룹 프로젝트에서 자료 조사 담당을 맡아 영어 원문 논문 2편을 요약·발췌하여 공유 문서로 정리함. 의견이 엇갈리는 지점에서 각 입장의 논거를 한 장으로 재정리해 합의 도출의 실마리를 제공함.

---

## 다. 3층 — 해석 (활동 → 의미)

**목적**: 관찰된 행동·산출물에서 **역량·태도·사고방식**을 추론.

### 해석 문장 패턴

- "이러한 과정에서 ~을 읽어내는 태도가 드러남."
- "단순한 ~을 넘어 ~까지 고려하는 사고방식이 관찰됨."
- "활동을 수행하는 방식에서 ~에 대한 감수성을 확인할 수 있음."
- "~을 통해 ~역량이 구체화되는 과정을 보임."
- "이러한 접근은 ~에 대한 깊은 이해 없이는 어려운 시도임."

### 해석할 수 있는 역량 예시 (구체 언어로)

| 관찰된 행동 | 의미화된 역량 |
|---|---|
| 여러 자료의 관점을 표로 정리 | 정보 구조화 능력, 비교 분석력 |
| 피드백 반영하여 글 재구성 | 메타인지적 자기 조정력 |
| 상대 논거 인정 후 재반박 | 논증의 유연성, 비판적 수용력 |
| 원문과 번역 대조 | 언어 간 뉘앙스 감수성 |
| 어휘 노트 자율 구축 | 자기주도 학습 시스템 설계력 |
| 지역·문화 맥락 덧붙임 | 맥락적 독해, 텍스트 외 지식 통합력 |

### 금지 사항
- "~잠재력이 크다" 같은 형용사 나열 금지
- 반드시 **활동과의 연결고리**가 문장 안에 있어야 함

---

## 라. 4층 — 전망 (미래지향)

**목적**: 현재 역량이 **어떤 영역·자질로 확장 가능한지**를 구체적 언어로 제시.

### 전망 문장 패턴

- "~에서 요구되는 ~역량의 토대가 되기에 충분한 모습임."
- "향후 ~상황에서 ~을 통해 ~을 수행할 잠재력이 기대됨."
- "~방향으로의 심화 탐구가 이루어진다면 ~에서 의미 있는 기여가 가능할 것으로 보임."
- "~에 대한 이해는 ~분야의 ~국면에서 유의미한 자산이 될 것으로 예상됨."

### 전망 영역의 스펙트럼 (넓게 → 좁게)

| 범위 | 예시 표현 |
|---|---|
| **자질 차원** | 맥락 해석력, 다관점 조정력, 질적 분석 감수성 |
| **활동 영역** | 학문적 글쓰기, 질적 연구, 비교 문화 탐구, 국제 협상 |
| **진로군** | 인문사회 탐구, 정책 분석, 언어 교육, 지역 연구 |
| ⚠️ **직업 단정** | (X) 의사, 변호사, 외교관 |

### 진로 정보가 있을 때 vs 없을 때

**있을 때**: 진로를 단정짓지 말고, **그 진로에서 필요한 역량 중 현재 드러난 것**을 연결.
> 공공정책 분야에 관심을 두고 있는 학생에게 이와 같은 다층적 텍스트 해석력은 정책 문서의 이해관계자 분석에 유효한 자산이 될 것으로 보임.

**없을 때**: 영역을 특정하지 말고 **전이 가능한 일반 역량**으로.
> 텍스트의 표면 논리와 함축 논리를 동시에 처리하는 능력은 인문·사회 전반의 탐구 활동에서 요구되는 기초 자질임.

## 마. 네 층위를 한 덩어리로 엮기 — 연결어 사전

층과 층을 잇는 전환어·종합어를 정교하게 사용하면 "엮음"의 품질이 올라갑니다.

| 전환 지점 | 연결 표현 |
|---|---|
| 1층 → 2층 | "이러한 성향은 ~활동에서 구체화됨.", "~학기 수업에서 이는 ~로 나타남." |
| 2층 → 3층 | "이러한 과정에서", "해당 활동을 수행하는 방식에서", "일련의 시도에서 드러나는 것은" |
| 3층 → 4층 | "이는 ~역량으로 수렴되며", "축적된 이러한 학습 태도는", "이러한 역량의 흐름은" |

## 바. 영어 원어·전문 어구 인용을 적극 활용할 것 (개성 증폭 기법)

세특이 **영어과** 특기사항이라는 점은, **영어 원어를 본문에 자연스럽게 삽입할 수 있다는 특권**이기도 합니다. 한국어만으로 서술된 세특은 읽을 때 어떤 학생에게나 해당될 수 있는 범용성에 빠지기 쉽지만, **구체적인 영어 어구가 본문에 인용되면 그 자체로 개성이 확보**됩니다.

### 어떤 영어 어구를 인용할 수 있는가

| 유형 | 예시 |
|---|---|
| **활동·프로젝트명** | Opinion Essay, 3분 English Speech, Scientific English Reading, Book Report |
| **텍스트 제목** | *The Kite Runner*, 'Digital Footprint and Me', BBC 6 Minute English |
| **핵심 논지·어구** | 'superposition을 여러 서랍이 열린 상태로 비유', 'means-tested welfare', 'creative subjectivity' |
| **전문 용어** | parameter selection, code-switching, hedging, information asymmetry |
| **토론 주제** | 'Should AI be used in classrooms?', 'Universal Basic Income' |
| **도구·플랫폼** | TED Talk, README, Reddit AMA, Wikipedia edit history |

### 인용의 원칙

- **따옴표 통일**: 작은따옴표('...') 사용 일관성 유지
- **이탤릭 대체**: 책·영화 제목은 따옴표 또는 *기울임* 대신 따옴표 권장 (NEIS 서식 한계)
- **과다 금지**: 500자 기준 영어 어구 인용은 **2~4회 정도가 적정**. 너무 많으면 가독성 저하
- **필수 맥락화**: 인용된 영어 어구가 *왜* 등장하는지 바로 이어지는 문장에서 맥락 설명

### 효과 비교

**약한 서술 (원어 없음)**:
> 2학기 영어 보고서에서 양자역학 개념을 설명하는 글을 작성하며 비유적 표현을 활용함.

**강한 서술 (원어 인용)**:
> 2학기 영문 보고서에서 'superposition'을 여러 서랍이 동시에 열린 상태로 치환한 비유를 통해 전문 개념의 소통 가능성을 실험함.

→ 같은 정보량이지만 후자는 **이 학생이 어떤 어휘를 어떤 방식으로 다뤘는지**가 즉각 전달됨.

## 사. 완성된 예시 (500자 모형)
> 텍스트를 읽을 때 표면 정보보다 필자의 숨은 가정을 먼저 찾는 독해 성향을 보임. 2학기 *공정무역* 주제 프레젠테이션에서 소비자와 생산자 관점을 대조하는 투 트랙 구조로 발표를 구성하고, 청중 질문에 즉흥적 사례를 추가하는 유연성을 보임. AI 생성 콘텐츠의 저작권을 다룬 Opinion Essay(350단어)에서는 '창작 주체성' 개념을 핵심 쟁점으로 설정하고, 피드백 이후 주장-근거 배치를 반박-재반박 구조로 전면 재조직함. 활동을 수행하는 방식에서 관점을 교차시켜 논증을 세우는 사고 습관과, 자신의 글을 타자의 시선으로 재독하는 메타인지적 조정력이 일관되게 관찰됨. 이러한 역량은 인문·사회 분야의 탐구 활동에서 요구되는 맥락 해석력과 논증 구성력의 토대가 되기에 충분하며, 심화된 쟁점 중심 읽기를 이어간다면 질적 분석이 요구되는 어떤 영역에서도 유의미한 성장이 가능할 것으로 기대됨.

**분석**:
- 1층: 첫 문장
- 2층: 두 활동 (프레젠테이션, 에세이)
- 3층: "활동을 수행하는 방식에서 ~관찰됨"
- 4층: "이러한 역량은 ~기대됨"
- 4기능: 읽기·말하기·쓰기 명시, 듣기는 "청중 질문 응답"에서 간접 함의
- 금기어: 없음
- 글자 수: 약 490자
