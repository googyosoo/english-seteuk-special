# 교사 입력 양식

세특 작성에 필요한 정보를 교사가 제공하는 두 가지 방식입니다. Claude는 **어떤 방식이든 받아서 내부에서 구조화**한 뒤 세특을 생성합니다.

---

## 방식 A: 자유 서술형 (Quick Input)

교사가 평소 쌓아둔 관찰 메모·평가 노트·수업 피드백을 **그대로 붙여넣는** 방식. 별도 구조화 없이 정보를 덤프해도 됨.

### 예시 (교사가 이렇게 입력해도 됨)
```
2학기 수행평가: 환경 주제 에세이. 
초안은 구조 엉성. 피드백 뒤 논거 배치 완전 재조직함.
모둠 토론에서 의장 역할 자주 맡음. 상대 의견 인정 후 재반박 잘함.
평소 어휘 노트를 주제별로 묶어 정리함. 자기주도적.
진로는 아직 미정. 인문사회 계열 관심.
공통영어1, 500자 정도로.
```

### Claude의 처리 방식
1. 이 메모를 5대 영역(성취수준·산출물·태도·의지·학습전략)에 매핑
2. 누락·불분명한 정보가 있으면 **1~3개만** 핵심 질문으로 되물음
3. 확인된 정보로 4층 구조 세특 생성

---

## 방식 B: 구조화 양식 (Full Input)

보다 체계적으로 정보를 제공하고 싶은 경우. 아래 양식을 복사해 빈칸을 채워 보내면 됨. **빈칸은 비워둬도 되며**, 비어 있는 영역은 서술에서 제외됨.

### 양식

```markdown
## [기본 정보]
- 과목명: (공통영어1 / 공통영어2 / 영어I / 영어II / 기타)
- 학년·학기: (예: 1학년 2학기)
- 분량: (기본 500자 / 별도 요청 시 지정)

## [활동 근거 — 최소 2개 이상]

### ① 활동
- 활동명/유형: (예: Opinion Essay 작성, 3분 Speech, 모둠 토론)
- 주제·소재: 
- 학생의 접근 각도·특징: 
- 산출물 내용: 
- 수정·발전 과정: 
- 특기사항: 

### ② 활동
- (동일 양식)

### ③ 활동 (선택)
- (동일 양식)

## [수업태도·학업의지·학습전략]
- 수업 참여 특징: 
- 질문·상호작용 양상: 
- 자기주도 학습 사례: 
- 학습 전략·시스템: 
- 피드백 수용·반영: 

## [진로·관심 정보 — 선택]
- 관심 분야: 
- 진로 방향: 
- 연결 짓고 싶은 역량: 

## [특별 요청 — 선택]
- 강조하고 싶은 포인트: 
- 피해야 할 표현: 
- 기타: 
```

## 입력 품질을 높이는 팁 (교사에게 안내할 것)

### 1. 활동은 "무엇을" 보다 "어떻게"가 중요
- ❌ "에세이를 잘 썼음"
- ⭕ "에세이 초안은 병렬 서술이었는데, 피드백 후 주장-근거-반박 구조로 전면 재조직"

### 2. 학생 고유의 관점/접근을 1개 이상
- ❌ "환경 주제로 발표함"
- ⭕ "환경 문제를 경제 구조 관점에서 재해석하는 접근을 택함"

### 3. 변화·발전의 시간 축 확보
- ❌ "발표를 잘함"
- ⭕ "초반 긴장 심함 → 연습 방식 전환 → 학기말 Q&A에서 여유로운 응답 가능"

### 4. 진로는 있으면 좋지만 없어도 됨
- 진로가 없어도 **인문사회 탐구 / 자료 분석 / 협상 / 언어 감수성** 같은 전이 가능한 역량으로 전망 서술이 가능

### 5. 금기 정보는 입력 단계에서 거르기
- 교외 수상, 학원 수강, 공인 시험 점수, 가정 배경 등은 입력해도 Claude가 제외함

---

## Claude의 후속 질문 원칙

입력이 부족할 때 Claude는 **한 번에 많이 묻지 않고**, 세특 품질에 결정적인 **1~3개 질문만** 선별해 되묻습니다.

**우선순위**:
1. 산출물(말하기·쓰기) 정보가 하나도 없을 때 → 최우선 질문
2. 활동은 있는데 학생 고유의 각도가 불분명할 때 → 접근 방식 질문
3. 성취수준 판단이 애매할 때 → 과제 난이도·결과 질문

**질문 예시**:
- "말하기 또는 쓰기 산출물 중 기억나는 특징(구조·관점·언어 사용)이 있을까요?"
- "이 활동에서 이 학생만의 독특한 시도가 있었다면 어떤 것이었나요?"
- "피드백 전후로 어떤 변화가 가장 인상적이었나요?"
