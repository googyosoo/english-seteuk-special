# 고등학교 영어과 세특(세부능력특기사항) 작성 스킬

2022 개정 교육과정 기반으로, 영어과 성취기준·4기능(듣기·말하기·읽기·쓰기) 활동·학습태도·학업의지·메타인지 학습전략을 **하나의 서술 안에 유기적으로 녹여내는** 세특을 생성합니다.

## 이 스킬이 해결하는 문제

교사 현장에서 세특은 두 가지 극단 사이에서 흔들립니다.

- **한쪽 극단**: "열심히 참여함, 영어 실력 우수" 같은 추상적 상찬 — 변별력 없음
- **반대쪽 극단**: 활동 나열만 가득한 불릿형 기록 — 학생의 얼굴이 안 보임

좋은 세특은 **활동(사실) → 과정(관찰) → 해석(의미) → 전망(가능성)** 의 네 층위가 한 덩어리의 서사로 흐르면서, 읽는 사람이 "이 학생이 누구인지"를 짐작하게 해야 합니다. 이 스킬은 그 흐름을 강제합니다.

## 핵심 원칙: 개성 × 발전성 × 미래지향

세 원칙은 독립적이지 않고 **같은 문장 안에서 동시에 작동**합니다.

### 가. 개성(Individuality) — 이 학생만의 것
- 활동명은 물론 **텍스트 소재·산출물의 관점·접근 방식**까지 구체화
- "영어 발표"가 아니라 "기후 난민 문제를 다룬 TED Talk를 요약·비평한 발표"
- **학생 고유의 해석 각도**를 명시 ("사회학적 관점에서", "비교문화적 시각으로")

### 나. 발전성(Development) — 시간 축의 변화
- 초반 → 후반의 변화, 피드백 수용 → 반영, 실패 → 재도전 궤적 포함
- "이미 잘함"이 아니라 "무엇이 어떻게 좋아졌는가"
- 단, 초반의 부족함을 서술할 때는 **부정어가 아닌 성장의 출발점**으로 표현

### 다. 미래지향(Prospective) — 잠재력의 구체화
- 추상적 칭찬("잠재력이 크다") 금지
- **현재 역량의 확장 가능성**을 구체적 영역과 연결
- 특정 직업 단정 대신 **역량·자질** 중심 (예: "질적 연구에서 요구되는 맥락 독해력")

→ 상세한 서술 기법은 `references/writing_templates.md` 참조.

## 워크플로우 (5단계)

### 1단계: 입력 정보 수집

교사는 **두 가지 방식** 중 하나로 입력할 수 있습니다.

**(A) 자유 서술형** — 메모·관찰 일지를 그대로 붙여넣기
예: "2학기 수행평가로 환경 주제 에세이 썼는데, 초안은 구조가 엉성했지만 피드백 받고 논거 배치 다시 함. 그리고 모둠 토론에서 의장 역할 맡았고 잘했음."

**(B) 구조화 양식** — 영역별로 정리된 입력
양식은 `references/input_templates.md`에 수록. 교사가 원하면 그 양식을 먼저 제시.

**확인해야 할 최소 정보** (부족하면 되물을 것):
1. 과목명 (공통영어1/2, 영어I/II 등)
2. 학기·차시 정보 (선택)
3. 수업 활동 중 **최소 2개** 이상의 구체적 근거
   - 말하기 또는 쓰기 산출물 **반드시 1개 이상**
   - 수업태도·학업의지·학습전략 중 **최소 1개 이상**
4. 학생의 관심 분야·진로 방향 (있으면 미래지향 서술에 활용, 없어도 됨)
5. 분량 (기본 500자, 요청 시 700~800자)

**근거가 불충분할 때**: 세특을 바로 쓰지 말고, "활동명은 있지만 산출물의 구체적 내용(어떤 관점/어떤 논거)이 없어서 개성이 드러나지 않을 수 있습니다. 다음 중 답해주실 수 있는 항목이 있을까요?" 형태로 되물을 것.

### 2단계: 총체적 영역 매핑

제공된 근거를 다음 **5개 영역**에 매핑합니다. 한 개 활동이 여러 영역에 걸쳐도 좋습니다.

| 영역 | 관찰 대상 | 서술 각도 |
|---|---|---|
| ① **성취수준** | 4기능 수행 수준 | 교육과정 성취기준 언어로 재서술 |
| ② **말하기·쓰기 산출물** | 발표·에세이·토론·프로젝트 | 구조·논거·관점·언어 사용 |
| ③ **수업태도** | 참여·질문·협력·몰입 | 구체적 에피소드 기반 |
| ④ **학업의지** | 자기주도성·지속성·도전성 | 변화 궤적 중심 |
| ⑤ **학습태도(메타인지)** | 자기평가·전략 수정·자료 활용 | 학습을 *어떻게* 하는가 |

**매핑 원칙**: 5개 영역을 모두 "빈칸 채우기"하려 하지 말 것. 제공된 근거에서 **자연스럽게 추출 가능한 3~5개 영역**만 골라 서사로 엮는다. 근거 없는 영역을 채우면 "~로 보인다" "~할 것이다"식 추측성 표현이 나옴 → 세특 품질 저하.

성취기준(2022 개정) 언어로의 변환은 `references/achievement_standards_2022.md` 참조.

### 3단계: 서술 구조 설계 (4층 구조)

500자 내외 세특을 다음 흐름으로 배열합니다.

```
[1층 — 진단] 학생의 영어과 접근 성향·강점 (1~2문장)
          ↓
[2층 — 전개] 구체적 활동 근거 (2~3가지, 과정·산출물 묘사)
          ↓
[3층 — 해석] 활동에서 드러난 역량·태도의 의미화
          ↓
[4층 — 전망] 발전 궤적과 미래지향적 잠재력
```

**주의**: 4층을 명시적인 단락으로 나누지 말 것. **한 덩어리의 유려한 서술**로 흐르게 하되, 내부적으로 이 층위를 다 품고 있어야 함.

층별 상세 기법과 문장 패턴은 `references/writing_templates.md` 참조.

### 4단계: 초안 작성

다음 체크리스트를 **쓰면서** 염두에 둘 것.

- [ ] 첫 문장이 학생 고유의 접근 성향을 드러내는가? (추상어 금지)
- [ ] 활동은 **무엇을·어떻게·그 결과**까지 드러나는가?
- [ ] 4기능 중 최소 2기능이 언급되는가?
- [ ] 변화·발전의 시간 축이 느껴지는가?
- [ ] 마지막 문장이 **구체적 역량**으로 미래를 그리는가?
- [ ] 금기 표현이 없는가? (`references/prohibited_expressions.md`)

**문체**:
- 종결형 "~함.", "~임." (개조식 아님, 서술형 문어체)
- 학생 이름·대명사(그/그녀) 사용 금지 — 단 **본인·자신·스스로**는 교사 관찰 관습어로 허용
- 교사 1인칭(나·필자) 금지
- 한 문장 길이는 80자 내외 권장, 최대 120자
- 영어 표현은 따옴표 없이 자연스럽게 삽입 가능

**분량 운용 원칙 (중요)**:
- 최종 목표가 500자 ±10%(450~550자)라면, **초안 작성 시 목표는 520~530자**로 타이트하게 시작할 것
- 초안이 느슨하게 쓰이면 글자수 초과 → 사후 압축 과정에서 리듬·개성이 훼손되기 쉬움
- 반대로 초안이 너무 짧으면(≤480자) 보강 과정에서 형용사나 상투어가 삽입될 위험

### 5단계: 자가 검증 및 출력

초안 완성 후, `assets/verification_checklist.md`의 **8개 검증 항목**을 모두 통과시키고 결과를 구조화하여 출력합니다.

출력 형식은 아래 "출력 형식" 섹션 참조.

## 작성 시 가장 자주 발생하는 실수 (필독)

1. **"성실함·열정·적극성" 같은 범용 형용사로 시작** → 어떤 학생에게도 해당됨. 반드시 구체 활동·접근 방식으로 첫 문장을 열 것.
2. **활동만 나열하고 해석이 없음** → "~을 발표하였고, ~을 작성함." 만으로 끝나면 관찰 기록에 불과. 반드시 3층(해석)과 4층(전망)을 붙일 것.
3. **"뛰어난 잠재력을 보임" 같은 추상적 미래 서술** → 어떤 분야의 어떤 역량인지 구체화.
4. **학생의 진로를 단정** → 특정 학과·직업 단정 대신 "질적 연구", "협상", "교차 분석" 등 전이 가능한 역량으로.
5. **성취기준을 그대로 복붙** → 성취기준은 교사용 언어. 세특에서는 학생의 수행을 통해 **구현된 모습**으로 재서술해야 함.
6. **금기어 간과** — 수상·대회·외부활동·특정 자격증명 등. 반드시 검증 단계에서 확인.
