# 세특 자가 검증 체크리스트

초안 작성 후 다음 8개 항목을 모두 통과시킵니다. 하나라도 미충족이면 해당 부분만 재작성.

---

## 1. 글자수·바이트 검증
- [ ] 공백 포함 글자수가 교사 요청 분량 ±10% 범위인가?
- [ ] NEIS 1500Byte를 초과하지 않는가? (한글 1자 ≈ 3Byte, 영문 1자 = 1Byte)
- [ ] 분량 부족으로 내용이 얕거나, 과다로 문장이 늘어지지 않는가?

**참고 계산**: 500자 내외 한글 = 약 1500Byte, 영문 섞을 경우 글자수 여유 생김

---

## 2. 첫 문장 — 개성 검증
- [ ] 첫 문장에 "성실·적극·열정" 같은 범용 형용사가 **없는가**?
- [ ] 학생 고유의 접근 방식·강점·사고 성향이 드러나는가?
- [ ] "이 첫 문장을 다른 학생에게 그대로 써도 어색하지 않다"면 재작성

**테스트**: 첫 문장에서 학생 이름을 지우고 반 학생 30명 중 몇 명에게 해당될까? 5명 이상이면 실패.

---

## 3. 활동 근거의 구체성
- [ ] 최소 2개 활동이 **활동명 + 주제 + 과정 + 산출물/결과** 수준으로 서술되었는가?
- [ ] "~을 잘했음", "~에 참여함" 같은 막연한 기술이 없는가?
- [ ] 학생 고유의 접근 각도가 최소 1회 이상 명시되었는가?

---

## 4. 4기능 통합
- [ ] 읽기·쓰기·듣기·말하기 중 **최소 2기능** 이상이 언급되는가?
- [ ] 기능별 언급이 단순 나열이 아니라 유기적으로 연결되는가?

**흔한 실수**: 쓰기만 반복 강조하여 다른 기능이 함의조차 안 됨. 발표 활동에서도 "듣기"와 "말하기" 양방향을 모두 담을 수 있음.

---

## 5. 발전·변화의 시간 축
- [ ] 초반→후반, 초안→최종, 피드백 전→후 중 **하나 이상의 궤적**이 드러나는가?
- [ ] 단순 "이미 잘함" 서술이 아닌가?

**약한 표현**: "꾸준히 노력함" → **강한 표현**: "초안의 X를 Y로 재구성함", "연습 방식을 A에서 B로 전환한 결과"

---

## 6. 해석과 전망
- [ ] 활동 서술 뒤에 **역량/태도로의 해석 문장**이 있는가?
- [ ] 마지막 부분에 **미래지향적 전망**이 구체적 영역·자질로 제시되었는가?
- [ ] "~큰 잠재력이 기대됨" 같은 추상적 전망이 아닌가?

**테스트**: 전망 부분만 떼어서 읽었을 때, 어떤 분야의 어떤 역량을 말하는지 명확히 느껴지는가?

---

## 7. 금기 표현 스캔
다음을 본문에서 검색하여 0건을 확인:
- [ ] 교외 수상·대회·경시 관련 단어 (수상, 대회, 경시, 입상)
- [ ] 공인 시험 이름 (TOEIC, TEPS, OPIc, IELTS, TOEFL)
- [ ] 학원·어학원·강사 언급
- [ ] 특정 교재 상표명 (수능특강, 수능완성, 출판사명 등)
- [ ] 가정 배경·경제 상황
- [ ] 타 학생과의 직접 비교 ("반에서", "또래 중")
- [ ] 특정 직업·학과 단정 ("~가 될", "~학과에 적합한")
- [ ] 과장 표현 ("완벽하게", "최고 수준", "반드시")

---

## 8. 문체·형식
- [ ] 종결 어미가 "~함.", "~임." 중심인가?
- [ ] 구어체·의문형·감탄형이 섞이지 않았는가?
- [ ] 학생 이름·대명사(그/그녀/그들) 사용이 없는가?
- [ ] 교사 1인칭(필자·나·저)이 없는가?
- [ ] ※ 본인·자신·스스로는 교사 관찰 관습어로 허용 (학생의 주체적 수행 부각 맥락에서 자연스러움)
- [ ] 개조식이 아닌 연결형 서술인가?
- [ ] 한 문단으로 이어지는가?

## 검증 실패 시 자동 재작성 규칙

| 실패 항목 | 재작성 범위 |
|---|---|
| ① 글자수 | 전체 — 분량 재조정 |
| ② 첫 문장 | 1층(진단)만 재작성 |
| ③ 활동 구체성 | 해당 활동 문장만 보강 (추가 정보 요청 가능) |
| ④ 4기능 | 2층(전개)에서 기능 언급 추가 |
| ⑤ 시간 축 | 해당 활동에 변화 궤적 삽입 |
| ⑥ 해석·전망 | 3층·4층 재작성 |
| ⑦ 금기 표현 | 해당 문장만 치환 |
| ⑧ 문체 | 어미만 일괄 수정 |

**원칙**: 문제가 있는 부분만 수술하되, 전체의 자연스러운 흐름을 유지할 것.
